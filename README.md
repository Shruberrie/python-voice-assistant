# Python Voice Assistant

This project implements a basic Python voice assistant that can listen to voice commands and perform simple tasks such as searching the web, opening applications, or providing information.

The assistant processes speech input using speech recognition and responds using text-to-speech.

## Features

- Voice command recognition
- Text-to-speech responses
- Wikipedia search
- Basic task automation
- Web queries

## Project Structure

src/
  skye.py  -> main voice assistant script

## Installation

Clone the repository:

git clone https://github.com/Shruberrie/Voice-Assistant.git

Install dependencies:

pip install -r requirements.txt

## Usage

Run the assistant:

python src/skye.py
